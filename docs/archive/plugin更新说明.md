## 🔧 使用方法

### 在 uTools 中使用

1. **打开游戏列表**
   - 输入: `小游戏` / `游戏`
   - 进入游戏集合主页

2. **快速打开特定游戏**
   - 俄罗斯方块: 输入 `俄罗斯方块` / `tetris`
   - 六边形俄罗斯方块: 输入 `六边形` / `hextris`
   - 小黑屋: 输入 `小黑屋` / `adarkroom`
   - 糖果盒子2: 输入 `糖果` / `candy`
   - 人生重开模拟器: 输入 `人生` / `重开`
   - 坦克大战: 输入 `坦克大战` / `tank`

### uTools 云存档功能

所有游戏都支持：
- ✅ 自动保存游戏进度
- ✅ 多设备同步存档
- ✅ 存档数据加密存储
- ✅ 旧存档自动兼容验证
