# 调试工具使用说明

## 🛠️ 开发者调试工具

为了方便在线上环境排查问题，本应用集成了 vConsole 调试工具。

### 快捷键

按下 **Ctrl+Shift+L** 即可打开或关闭调试控制台。

### 功能说明

vConsole 是一个轻量级的移动端调试工具，提供以下功能：

#### 1. Console 控制台
- 查看所有日志输出
- 查看 JavaScript 错误
- 执行 JavaScript 代码

#### 2. System 系统信息
- 查看浏览器信息
- 查看用户代理
- 查看屏幕信息

#### 3. Network 网络请求
- 查看所有网络请求
- 查看请求头和响应
- 查看请求耗时

#### 4. Element 元素检查
- 检查页面元素
- 查看 CSS 样式
- 查看元素属性

#### 5. Storage 存储查看
- 查看 localStorage
- 查看 sessionStorage
- 查看 Cookie

### 使用场景

#### 场景1: 查看游戏报错
```
1. 进入游戏
2. 按 Ctrl+Shift+L 打开控制台
3. 查看 Console 标签页中的红色错误信息
4. 截图或复制错误信息反馈给开发者
```

#### 场景2: 查看存档数据
```
1. 按 Ctrl+Shift+L 打开控制台
2. 切换到 Storage 标签页
3. 查看 localStorage 中的游戏存档
4. 检查存档数据是否正确
```

#### 场景3: 查看网络请求
```
1. 按 Ctrl+Shift+L 打开控制台
2. 切换到 Network 标签页
3. 进行游戏操作
4. 查看是否有失败的请求
```

### 注意事项

1. **默认隐藏**: 调试工具默认隐藏，不会影响正常游戏体验
2. **快捷键独占**: Ctrl+Shift+L 是调试专用快捷键，游戏中未使用
3. **数据安全**: 请勿在公共场合打开调试工具，避免泄露敏感信息
4. **性能影响**: 调试工具会占用少量内存，建议问题排查后关闭

### 常见问题

#### Q: 快捷键没有反应？
A: 请确保：
- 已经进入游戏界面
- 同时按下 Ctrl、Shift、L 三个键
- 如果是新游戏，等待几秒让 vConsole 初始化完成

#### Q: 看不到绿色的切换按钮？
A: 这是正常的！为了保持界面干净，我们隐藏了默认按钮，只能通过快捷键控制。

#### Q: 调试工具会影响游戏性能吗？
A: 影响很小。vConsole 占用约 287KB 内存，默认隐藏时几乎无性能影响。

#### Q: 如何完全禁用调试工具？
A: 当前版本在所有环境都启用了调试工具。如果需要禁用，请联系开发者。
